## Sorun: Hard-set offset, ekstrapolasyonu patlatıyor

Her üç erken çıkış dalında `Sunucu_Zaman_Farki`'nı hard-set yapıyorum. Bunun ne yaptığını izleyelim:

### Senaryo: Aynı sunucu zamanı (pause), 200ms aralıklarla paket

```
t=0ms:    Paket gelir, offset = perf.now()/1000 - serverTime → hard set
          → dtSincePacket = estimatedServerNow - serverTime ≈ 0
          → ekstrapolasyon = 0, uçak Son_Gercek_Konum'da

t=100ms:  Render çağrılır, perf.now ilerledi
          → estimatedServerNow = (perf.now+100ms)/1000 - offset → ≈ serverTime + 0.1
          → dtSincePacket ≈ 0.1s → uçak 0.1×hız kadar İLERİ gider

t=200ms:  Aynı zaman paketi gelir, offset TEKRAR hard-set edilir
          → estimatedServerNow tekrar ≈ serverTime
          → dtSincePacket ≈ 0 → uçak GERİYE çekilir!

→ İLERİ-GERİ = TİTREME! 🔴
```

Aynı sorun "aynı konum" dalı için de geçerli.
